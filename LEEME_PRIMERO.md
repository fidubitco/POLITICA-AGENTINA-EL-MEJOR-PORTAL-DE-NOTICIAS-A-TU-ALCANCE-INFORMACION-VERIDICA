## 🆘 ¿NECESITAS AYUDA?

### Si algo no funciona:

1. **Error de build**
   ```bash
   pnpm install
   pnpm prisma generate
   pnpm build
   ```

2. **Error de database**
   - Verificar `DATABASE_URL` en Vercel
   - Verificar que copiaste completa la variable

3. **No se generan noticias**
   - Verificar `OPENAI_API_KEY` en Vercel
   - Verificar que OpenAI tenga crédito

4. **Otros problemas**
   - Ver logs en Vercel Dashboard
   - Revisar `HAZLO_TU_MISMO_FACIL.md`
